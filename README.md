# Smart Study Tracker

Smart Study Tracker is a simple Python desktop application that helps students record and analyze their study sessions. Users can add study sessions, store the data in a database, view study history, and visualize study time using graphs.

## Features

- Add study sessions with subject and minutes studied
- Store study data using SQLite
- View all study sessions in a table
- Show total study time
- Visualize study time with graphs
- Simple and clean GUI built with Tkinter

## Tech Stack

- Python
- Tkinter (GUI)
- SQLite (Database)
- Matplotlib (Data Visualization)

## How to Run

1. Clone the repository

```
git clone https://github.com/yourusername/smart-study-tracker.git
```

2. Go to the project folder

```
cd smart-study-tracker
```

3. Install required library

```
pip install matplotlib
```

4. Run the application

```
python main.py
```
